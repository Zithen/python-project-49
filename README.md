### Project Description:
#### Set of terminal games:

* ##### Requirements:
  * Python >3.10
  * PIP >23.x
  * Poetry >1.4.x

* ##### Installation:
  * Clone the repo
  * make build
  * make package-install
  
* ##### Games list:
  1. ###### Odd or Even
    * Run: brain-even
